# Mermaid Diagrams

A reference site showcasing every free [Mermaid.js](https://mermaid.js.org) diagram type with rendered examples and real-world software development use cases.

## Stack

- React 18 + TypeScript + Vite
- Tailwind CSS v4
- React Router v6
- Mermaid.js 11

## Development

```bash
pnpm install
pnpm dev
```

## Charts

20 diagram types across 5 categories — Flowchart, Sequence, Class, State, ER, User Journey, Gantt, Pie, Quadrant, Requirement, Gitgraph, C4, Mindmap, Timeline, Sankey, XY Chart, Block, Packet, Kanban, and Architecture.
